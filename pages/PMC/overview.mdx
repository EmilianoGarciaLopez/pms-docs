
# Overview of the PMC-C-CMX


<div className="nx-hidden">
## Overview
</div>

The PMC-C-CMX is a powerful control board designed for the Raspberry Pi Compute Module 4 or 5 (CM4/CM5). It offers a range of features and capabilities, making it an ideal choice for industrial control applications.

Key features include:
- Flexible powering options
- High-performance compute module
- Real-time control with STM32H7
- Expandable modular bus

Explore the sections below to learn more about each aspect of the PMC-C-CMX controller.
